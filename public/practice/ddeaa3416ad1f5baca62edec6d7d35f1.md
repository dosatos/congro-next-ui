
## Ad Serving System

**Question:** Design an ad serving system for a large-scale advertising platform

**Hints:**
- Consider implementing a real-time bidding system for ad auctions
- Think about how to handle user profiling and ad targeting
- Explore options for minimizing ad serving latency
- Consider implementing fraud detection mechanisms
- Think about scaling the system to handle billions of ad requests per day

> #### Key Considerations:
> - Real-time bidding
> - Ad targeting and personalization
> - Performance and latency
